Project information

  • 주제: AWS Cloud를 이용한 Hybrid Cloud 구축
  • 프로젝트 기간: 2024년 4월 15일 ~ 4월 24일
  • 핵심목표: Multi-Cloud구축 및 Hybrid Cloud 구축
  • 담당 역할: Project Leader
    AWS 서울, 오레곤 Reion 전체 구축
    Private 영역 WEB-WAS-DB 3-Tier 구축
    Multi-Region간 Peering
    RDS Proxy 구축 & 이중화

프로젝트 목표

  • Multi-Cloud를 구축하여 HA 구성
  • 트래픽 분산을 위한 Auto-Scaling Group과 ELB 구축
  • 웹 보안 및 트래픽 필터링, 차단을 위한 WAF-ACLs 구축
  • NAT기능을 통해 내/외부 간 통신은
    가능케 하되 직접적인 연결은 차단한다
  • On-Premise 구축하여 AWS Cloud와 VPN 연결
  • 리전간 Peering과 EFS를 통한 동일 데이터 공유

    • 배운점

    • AWS 클라우드의 Network Infra 구조
    • AWS Site-to-Site VPN 연결
    • RDS MariaDB 멀티 리전 읽기 전용 복제생성 방법
    • Route53을 통한 DR구조 구현 방법
    • CloudFormation의 작동 방법 및 구축 방법

    어려운점

    • 보안그룹 설정시 Inbound와 Outbound 방법
    • RDS 멀티 리전 이중화 방법
    • Multi-Region간 동일 WEB Page 반응

    해결방법
